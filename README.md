# Орден обітниць

Особистий трекер стріків (чистих днів) по обітницях. Статичний сайт + PWA: ставиться на головний екран iPhone, працює офлайн. При першому відкритті за день — ритуал: дата → цитата → вшанування круглих дат → табло обітниць. Тап у будь-якому місці пропускає анімацію.

## Викласти на GitHub Pages
1. Створи репозиторій (новий або заміни старий `focus`).
2. Завантаж **усі файли в корінь** репозиторію (не в підпапку).
3. Settings → Pages → Source: **Deploy from a branch** → гілка `main`, папка `/ (root)` → Save.
4. За хвилину відкрий `https://<нік>.github.io/<репо>/` у Safari → Поділитися → **На екран «Додому»**.

## Приватність
Якщо репозиторій публічний — код і цитати видно будь-кому. Хочеш сховати — зроби репозиторій **Private** (Pages працює і так на безкоштовному акаунті). Назви обітниць лишаються справжніми в самому додатку.

## Що редагувати
Майже все — у `data.js`:
- `VOWS` — обітниці та стартові дати (дати також можна міняти просто в додатку, тапнувши обітницю);
- `QUOTES` — 50 цитат ритуалу;
- `MILESTONES` + `MILESTONE_MSGS` — круглі дати та привітання.

## Оновлення
Дані живуть локально в браузері (localStorage), окремо на кожному пристрої. Скидання стріку → тап по обітниці → «Зрив». Після зміни файлів **зміни версію кешу** в `sw.js` (`ordo-v1` → `ordo-v2`), інакше Safari покаже стару версію з кешу.
